# TODO List App using Shared Preferences
-This is a simple todo-list app using Shared Preferences in flutter to store data locally.

# Features
- Create
- Add
- Update
- Delete

# Dependencies
-(sdk: flutter)
-shared_preferences: ^2.0.8
-provider: ^6.0.5
